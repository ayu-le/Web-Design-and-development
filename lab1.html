Facts

The chemical formula for water, a crucial component in our search for life, is H2O. This is example of a subscript element.

To express an area of space, we might use a value like 105 light-years squared, demonstrating a superscript element.

Code Example

To define a paragraph in HTML, we use the tag: <p>.
